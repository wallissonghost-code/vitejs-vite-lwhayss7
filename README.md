# GXST Vibes

App de vídeos curtos estilo feed vertical, criado em React + Vite.

## O que já tem na V1

- Feed vertical com rolagem tela cheia
- Vídeos com autoplay, legenda, música e hashtags
- Curtir, salvar, comentar, compartilhar e seguir
- Aba Buscar com filtro por usuário, legenda, música e hashtag
- Aba Inbox com notificações demonstrativas
- Perfil com estatísticas e grid de vídeos
- Publicação demonstrativa usando URL de vídeo `.mp4`
- Dados salvos no `localStorage` do navegador

## Rodar no Replit

```bash
npm install
npm run dev
```

Depois abra a URL gerada pelo Replit.
